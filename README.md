# Calculadora

Aplicación de calculadora con operaciones básicas construida con Next.js y Tailwind CSS.

## Funcionalidades

- Suma, resta, multiplicación y división
- Encadenamiento de operaciones
- Soporte para decimales
- Teclas de teclado
- Interfaz visual moderna

## Instalación

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
npm start
```
